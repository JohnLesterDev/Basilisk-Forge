## üöÄ Getting Started

### Prerequisites
- Java Development Kit (JDK) 16
- Gradle 7.x or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Obelisk-Forge.git
   ```
2. Navigate to project directory:
   ```bash
   cd Obelisk-Forge
   ```
3. Build the project:
   ```bash
   ./gradlew build
   ```
4. Run the application:
   ```bash
   ./gradlew run
   ```
